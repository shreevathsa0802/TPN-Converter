\# TPN - Two-Port Network Converter



A professional desktop application for converting between different two-port network parameters.



\## Features

\- Convert between Z, Y, H, ABCD, and G parameters

\- Handle complex numbers

\- User-friendly GUI interface

\- Save/Load configurations

\- Export results

\- Windows executable available
